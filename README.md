## File Listing Utility

A Python script to list files and directories in this repository.

### Usage

```bash
# List files in the current directory
python3 list_files.py

# List files in a specific folder
python3 list_files.py "PDF AND MORE FILES"
python3 list_files.py "LS25P20195_Court_Bundle_Father"
```

### Features

- Lists all directories and files in a specified folder
- Displays file sizes in human-readable format
- Organized output with directories listed first
- Summary statistics (total directories and files)
